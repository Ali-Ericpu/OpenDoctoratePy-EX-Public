# DoctoratePy-EX-Public

本项目旨在作为DoctoratePy的分支，拓展与完善功能，实现类似SPT-AKI项目的单人游玩本地服务

公开版仓库不提供任何游戏数据或与客户端修改相关的内容

开发仅为个人兴趣

此项目禁止用于以任何形式的盈利目的进行分发或二次开发

## 可游玩内容

| 内容 | 备注 |
|:---:|:---:|
| 战斗 | 大部分可游玩 |
| 肉鸽 | 不支持随机内容 |
| 邮件 | 仅用于防止误触后404 |
